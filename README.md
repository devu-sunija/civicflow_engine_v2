# CivicFlow AI: Event-Driven Congestion Control Room

An intelligent traffic orchestration engine designed for the Flipkart GRiD Architecture Challenge. This system maps real-world event parameters directly to localized geohash matrices to dynamically calculate crowd density, flow saturation, and optimal logistical resource deployment.

## 📁 Project Architecture
* `backend.py` - High-performance FastAPI computation engine utilizing relative dataset resolution.
* `frontend.py` - Streamlit interactive operations dashboard with localized Folium mapping.
* `traffic_data.csv` - Historical structural dataset node repository.
